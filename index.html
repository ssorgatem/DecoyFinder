DecoyFinder

A Tool for finding decoy molecule sets for given active ligand sets


DecoyFinder is a graphical tool which helps finding sets of decoy molecules for a given group of active ligands. It can do so by finding molecules which have a similar molecular weight to the known actives (fast mode) or by finding molecules with a similar number of rotational bonds, hydrogen bond acceptors, hydrogen bond donors, logP value and molecular weight, but chemically different, which is defined by a maximum Tanimoto value threshold between active ligand and decoy molecule MACCS fingerprints (slow mode). Optionally, a maximum Tanimoto value threshold can be set between decoys in order assure chemical diversity in the decoy set in the second mode.

Install

  • Windows users: just use the installer, all needed files are provided.
  • Ubuntu 12.04 and Debian users: simply install the provided .deb package, all dependencies will be automatically installed by the package manager. If you open the .deb package using the software center, it may complain about the package being of bad quality and advice against its installation. Just ignore the warning. This will be fixed in future releases.


If you are running an OS for wich there are no DecoyFinder packages available, you must make sure you have all the dependencies are correctly installed and then it should work from the generic (source) package, just executing the following from a command line:

$ python PATH/TO/DECOYFINDER/decoy_finder.py

Dependencies

  • Python >= 2.5,  < 3
  • RDKit
  • OpenBabel with python bindings (optional, for extra file formats)
  • Cinfony
  • PyQt4 >= 4.6
